W05 Project: Website Planning Document

Green Acres Community Gardens

4. Site Name

Name: Green Acres Community Gardens

Reason: The name is friendly, evocative of fresh air and growth, and clearly identifies the site's function (a place for community gardening).

5. Site Purpose

The primary purpose is to be the central hub for the garden program. It will provide two main services:

  • Information and Recruitment: Attract new members and volunteers by providing detailed guidelines, contact information, and inspiring content about the benefits of community gardening.
  • Coordination and Events: Offer a dynamic, filterable list of upcoming planting workshops, cleanup days, and community events to coordinate participation among existing members.

6. Scenarios (Target Audience Users)

  1. **Scenario 1 (New User):** "I'm interested in renting a plot for the first time. Where can I find the rental fees and the rules?"
  2. **Scenario 2 (Existing Member):** "I have a free Saturday and want to volunteer. Which upcoming events are specifically focused on 'cleanup' activities?"

7. & 8. Color Schema

This plan uses the defined colors in this document.

Color Name Hex Code Usage Sample
Primary (Earth Green) #4CAF50 Header backgrounds, navigation buttons, important calls-to-action (CTAs).
Secondary (Rich Soil Brown) #8B4513 Footer background, primary text on light backgrounds, card borders.
Accent (Sky Blue) #87CEEB Hover states for links, weather display background, highlighting filtered content.
Light (Page Background) #FAFAFA Main background, text on dark backgrounds.

9. & 10. Typography

This plan uses the defined font(s) in this document.

Element Font Style Usage
Headings (H1, H2, H3) 'Playfair Display', serif Used for all page titles and section headings to provide an organic, traditional feel.
Body Text, Paragraphs, Lists 'Roboto', sans-serif Used for all body content, navigation, and detailed information for maximum legibility.

1. Naming Conventions

All files and folders will follow standard course naming conventions (lowercase, hyphenated):

  • **HTML:** index.html, about.html, events.html, siteplan.html
  • **CSS:** styles/base.css, styles/medium.css, styles/large.css
  • **JavaScript:** scripts/main.js
  • **Images:** images/ folder (e.g., images/garden-map.jpg)
